# Control Flow

```java
/** 基本控制语句 */
if (cond1){statements}else if (cond2){statements}...else{statements}
while (cond){statements}
do {statements} while (cond)
for (init;cond;control){statements}
switch (cond){case val1:statementes;break;...;default:...;break;}
for (int element:collection) statements
//可以是enum,char,byte,int,short,对>Java 7可以是String
```
