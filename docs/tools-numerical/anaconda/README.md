# Anaconda

## Introduction

conda 是但不限于：

* 虽然主要用于 python，但实际上似乎支持众多语言的包管理工具；
* 一个虚拟环境的管理工具。

Anaconda 是一个 python 和 R 的 distribution，它包含了 conda、python 和一些常用的科学计算包。
